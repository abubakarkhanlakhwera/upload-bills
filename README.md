# Water Bills Auto Uploader

A Chrome extension for scheduled water bill file uploads.

## Purpose

Uploading billing files repeatedly can waste time when the same page flow is used again and again. This extension explores automating file selection/upload steps and tracking scheduled upload runs.

## What This Shows

- Chrome extension automation
- Scheduled upload workflow
- Popup UI and content-script behavior

## Stack

- JavaScript
- Chrome Extension APIs
- HTML/CSS

## Status / Safety Note

Do not commit real billing files, portal credentials, or customer data.
